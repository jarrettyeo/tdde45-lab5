40 """
41 (len, point, neg) = Grisu.grisu(v::AbstractFloat, mode, requested_digits, [buffer], [bignums])
42
43 Convert the number `v` to decimal using the Grisu algorithm.
44
45 `mode` can be one of:
46 - `Grisu.SHORTEST`: convert to the shortest decimal representation which can be "round-tripped" back to `v`.
47 - `Grisu.FIXED`: round to `requested_digits` digits.
48 - `Grisu.PRECISION`: round to `requested_digits` significant digits.
49
50 The characters are written as bytes to `buffer`, with a terminating NUL byte, and `bignums` are used internally as part of the correction step. You can call `Grisu.getbuf()` to obtain a suitable task-local buffer.
51
52 The returned tuple contains:
53
54 - `len`: the number of digits written to `buffer` (excluding NUL)
55 - `point`: the location of the radix point relative to the start of the array (e.g. if
56 `point == 3`, then the radix point should be inserted between the 3rd and 4th
57 digit). Note that this can be negative (for very small values), or greater than `len`
58 (for very large values).
59 - `neg`: the signbit of `v` (see [`signbit`](@ref)).
60 """
